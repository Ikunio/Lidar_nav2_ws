# 启动说明书
### 一、仿真
#### 1. 建图
``` bash
1. 运行
./mapping_sim.sh

2. 保存地图
./save_map.sh

3. 保存点云
./save_pcd.sh
将点云文件移动到/src/me_nav2_bringup/pcd
```

#### 2. 导航
``` bash
1. 进入 /src/small_gicp_relocalization/launch/
修改launch中的prior_pcd_file，地址指向建好的pcd点云图

2. 进入 /src/me_nav2_bringup/launch/my_nav2_launch.py
修改launch中的map_yaml_file，指向建好的地图yaml文件

3. 启动
./nav2_sim.sh
```
